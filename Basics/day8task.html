Why does the Link component turn into an anchor tag, and className into class, when inspecting the DOM?

The Link component (from libraries like react-router-dom) gets rendered as an anchor tag () in the DOM through JSX transformations. React internally uses JSX to generate actual HTML elements and map attributes like "to" into "href" in the resulting tag. Regarding the className attribute: React uses "className" in JSX because "class" is a reserved keyword in JavaScript. During the rendering process, React modifies "className" to "class" using DOM APIs such as setAttribute() to ensure valid HTML output.
